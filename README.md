# 🚀 End-to-End DevOps Automation Project

## 📌 Project Overview

This project demonstrates a complete DevOps workflow using Docker, Docker Compose, Nginx, and GitHub.  
The application is containerized and deployed using a multi-container architecture with automated service management.

---

## 🛠️ Technologies Used

- Docker
- Docker Compose
- Nginx
- Node.js
- GitHub
- Git

---

## ⚙️ Features

✅ Dockerized Node.js application  
✅ Multi-container deployment using Docker Compose  
✅ Nginx Reverse Proxy configuration  
✅ Automated container orchestration  
✅ Lightweight and scalable architecture  
✅ DevOps workflow implementation  
✅ Easy deployment and portability

## ▶️ Run Project Locally

### Step 1 — Clone Repository

```bash
git clone https://github.com/ernikhil5101/devops-automation-project.git
```

### Step 2 — Move into Project Directory

```bash
cd devops-automation-project
```

### Step 3 — Build and Run Containers

```bash
docker compose up --build
```
